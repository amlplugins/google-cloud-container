# @amlplugins/google-cloud-container

Native `@google-cloud/container` SDK re-exported under the `@amlplugins` namespace.

> Google Kubernetes Engine.

## Install

```bash
npm install @amlplugins/google-cloud-container
```

## Use

```ts
import { /* named exports from @google-cloud/container */ } from "@amlplugins/google-cloud-container";
// or
import sdk from "@amlplugins/google-cloud-container";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
